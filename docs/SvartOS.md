# SvartOS

SvartOS is a collection of utils and apps to use effectively your computer.

Apps:
sddm
hyprland
quickshell
tofi

kitty
nautilus

wifi (nmcli)
bluetooth (bluez)
usb daemon


sddm:
```shell
sudo pacman -S sddm
sudo systemctl enable sddm
```

Hyprland:
```shell
sudo pacman -S hyprland
```
